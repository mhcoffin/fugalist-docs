    Expression-map Builder

    Expression-map builder #

    The Fugalist expression-map builder is an on-line tool for creating expression maps for Dorico. The intent is to simplify and streamline expression-map creation.

    Fugalist stores your data in the cloud, so to use it you have to be on-line and signed in to Fugalist.

    To get started, click on the project list icon in the Fugalist header:

    You will see a list of your projects. Click on + and enter a name to add a new project.
